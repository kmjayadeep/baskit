CI Summary

  • .github/workflows/build-apk.yml builds and tests app changes.
  • Release job runs on tags and creates APK + AAB artifacts.
  • .github/workflows/deploy-pages.yml publishes pages/ to GitHub Pages when pages/** changes.

Publishing Docs Updates

  1. Edit files under pages/.
  2. Open PR with summary and validation commands.
  3. Merge to main to trigger Pages deployment.
  4. Confirm legal page URLs remain stable for store compliance.
