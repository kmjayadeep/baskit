Local Development Setup

Prerequisites

  • Flutter SDK 3.35.3
  • Firebase project for Auth and Firestore
  • Android Studio and/or Xcode

Initial Setup

git clone <repository-url>
cd baskit/app
flutter pub get

Then add Firebase platform files:

  • app/android/app/google-services.json
  • app/ios/Runner/GoogleService-Info.plist (if not already present in local environment)

Run App

cd app
flutter run
