Current Product Scope

Baskit is a guest-first shopping list app with optional account sign-in for real-time collaboration and sync.

  • Use immediately in guest mode with no registration.
  • Store data locally by default, with migration to cloud after sign-in.
  • Share lists with granular permissions when authenticated.
  • Support iOS, Android, web, and desktop builds from one Flutter codebase.
