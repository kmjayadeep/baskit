# Baskit PRD Index

## Purpose
This index defines the required scope and order of product requirements. Use these files in order to recreate the full app behavior and architecture.

## Order
1. `prds/01-overview.md` - Product goals, platforms, screens, and core flows
2. `prds/02-authentication.md` - Guest-first auth modes and account behavior
3. `prds/03-storage-and-sync.md` - Local/Cloud routing, migration, sharing, and sync
4. `prds/04-firestore-data-model.md` - Firestore collections, fields, and indexes
5. `prds/05-state-management.md` - Riverpod provider and state requirements
6. `prds/06-ui-and-assets.md` - UI behavior, dialogs, and asset usage
7. `prds/07-development.md` - Build, test, and operational requirements
